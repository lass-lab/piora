# Script for Cosmos

## How to use
### Arguments Description
`<path_to_xsct> ./cosmos_init.tcl --help`

### Example
```bash
<path_to_xsct> ./cosmos_init.tcl     \
            -e ./cofing/app0.elf     \
            -b ./cofing/OpenSSD2.bit \
            -h ./cofing/system.hdf   \
            -i 127.0.0.1             \
            -p 3121
```
